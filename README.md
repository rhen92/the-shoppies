## Technology Used
- `JavaScript`
- `HTML`
- `CSS`

## Overview
Application that allows a user to search for movies and nominate them for an award. Timeboxed to 3 hours.

## Set-up
1. Clone this repo
2. Type `cd the-shoppies` to move into the directory
3. Type `open index.html` to see the app in action
